A wireless sensor node uses an antenna of length $L = 3\text{ cm}$. Assume $L = \lambda/4$ for efficient radiation. What carrier frequency should be used?
  • Find wavelength: $$\lambda = 4L = 4 \cdot 0.03 = 0.12\ \text{m}$$
  • Find frequency using $f = c/\lambda$, $c = 3\times10^8\ \text{m/s}$: $$f = \frac{3\times10^8}{0.12} = 2.5\times10^9\ \text{Hz}$$
  • $$\boxed{f = 2.5\ \text{GHz}}$$

Name and briefly explain the three main wireless propagation effects. Which one can cause a receiver to get a signal even without line-of-sight?
  • Reflection: Signal bounces off a large object (much larger than wavelength); can cause phase shift and constructive/destructive interference.
  • Diffraction: Signal bends around edges/obstacles.
  • Scattering: Signal is dispersed by objects with size comparable to wavelength.
  • The effect that allows reception without LOS: $$\boxed{\text{Diffraction}}$$

Why can QAM achieve better bandwidth efficiency than simple AM?
  • QAM combines two orthogonal carriers (I and Q channels).
  • Two sub-streams can be transmitted in parallel over the same channel bandwidth.
  • This doubles the spectral efficiency compared to simple AM, which uses only one carrier dimension.

Given $\dfrac{P_{rx}}{P_{tx}} = \dfrac{A_{tx}A_{rx}G_{tx}G_{rx}}{d^2}$, show how to separate the expression into a design-time term and a runtime term using logarithms.
  • Take $10\log_{10}$ on both sides: $$10\log_{10}\!\left(\frac{P_{rx}}{P_{tx}}\right) = 10\log_{10}(A_{tx}) + 10\log_{10}(A_{rx}) + 10\log_{10}(G_{tx}) + 10\log_{10}(G_{rx}) - 20\log_{10}(d)$$
  • Static (design-time): $$10\log_{10}(A_{tx})+10\log_{10}(A_{rx})+10\log_{10}(G_{tx})+10\log_{10}(G_{rx})$$ Antenna sizes and nominal gains — fixed during design.
  • Dynamic (runtime): $$-20\log_{10}(d)$$ Distance $d$ — changes at runtime.
